# Wanderlust – Full Stack Property Listing Platform

Wanderlust is a full-stack web application for managing travel and property listings. Users can create, edit, delete, search, and explore different property listings through a responsive and user-friendly interface.

##  Features

- **Authentication & Authorization:** Implemented secure user signup, login, and logout functionality with role-based access control for managing property listings.
- **Property Listing Management:** Developed complete CRUD operations for creating, viewing, updating, deleting, and searching rental property listings with dynamic database integration.
- **Review & Rating System:** Enabled authenticated users to add reviews and ratings for listed properties.
- **MVC Architecture:** Structured the application using the Model-View-Controller (MVC) architecture for clean, scalable, and maintainable code organization.
- **Security & Session Management:** Implemented password hashing, secure authentication, and session management to enhance application security.
